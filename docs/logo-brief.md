# Orthrus Logo Brief

## Concept

Orthrus — the two-headed dog from Greek mythology. Each head sees the same world differently, which maps to the project's dual-headed GUI testing approach.

## Recommended Sizes

| Use Case | Size | Format |
|----------|------|--------|
| Primary logo (source) | 1024x1024px | SVG preferred, PNG fallback |
| GitHub/GitLab avatar | 512x512px | PNG |
| README display | 256x256px | PNG |
| Favicon | 32x32px, 16x16px | ICO or PNG |
| Social/og-image | 1200x630px | PNG (logo centered on background) |
| Badge/icon | 128x128px | PNG |

## Guidelines

- **Square aspect ratio** for the primary logo — most platforms crop to square
- **SVG source** if possible — scales cleanly to any size
- **Simple enough to read at 32x32** — fine detail gets lost at small sizes
- **Works on both light and dark backgrounds** — consider a version with and without background, or a design that works on both
- Export at 2x for retina/HiDPI displays (e.g., 1024px for a 512px display target)

## File Locations

Place finished assets in `docs/assets/`:
- `orthrus-logo.svg` — vector source
- `orthrus-logo-1024.png` — high-res raster
- `orthrus-logo-512.png` — GitHub/avatar size
- `orthrus-logo-128.png` — badge/icon size
- `favicon.ico` — browser favicon
